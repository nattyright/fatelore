By Region

            

              

                

                  
Servants from the Same Region

                  
If a Servant isn't showing up, consider switching between country name & nationality (e.g. France, French). Otherwise, blame FGO for grouping Servants by continents (e.g. Rome <=> Europe) instead of countries of origin :)

                

                

              

            


            
By Alignment

            

              

                

                  
Servants with the Same Alignment

                  
Because Fate is a mess, the following alignments are interchangeable: Lawful Neutral <=> Lawful Balanced, True Neutral <=> Neutral Balanced, Chaotic Neutral <=> Chaotic Balanced :)
